# Swiggy Delivery Time Prediction

## Overview

This project predicts food delivery time using Machine Learning techniques.

The objective is to improve delivery estimation accuracy and enhance customer satisfaction.

---

## Business Problem

Food delivery platforms need accurate delivery time estimates.

Incorrect estimates can lead to:

- Poor customer experience
- Delivery delays
- Operational inefficiencies

This project aims to predict delivery time based on historical delivery data.

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## Workflow

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Building
5. Model Evaluation
6. Insights

---

## Key Insights

- Traffic conditions significantly impact delivery time.
- Distance is a strong predictor.
- Weather conditions affect delivery efficiency.
